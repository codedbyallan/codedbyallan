## Projeto em destaque

### TaskFlow — Gerenciador Inteligente de Tarefas

Aplicação fullstack para gerenciamento de tarefas, criada com **React, Vite, ASP.NET Core Web API, MongoDB Atlas, Docker, xUnit e Render**.

O projeto começou como um CRUD simples e evoluiu para uma aplicação completa, com frontend moderno, backend publicado, banco em nuvem, testes automatizados, Docker, Docker Compose, filtros, busca, agrupamento por prazo e documentação final com prints.

#### Principais recursos

- CRUD completo de tarefas;
- busca por título e descrição;
- filtros por status e prazo;
- tarefas atrasadas, para hoje, próximas, futuras, sem prazo e concluídas;
- edição inline;
- data de vencimento opcional;
- indicadores visuais de prazo;
- backend em ASP.NET Core Web API;
- persistência com MongoDB Atlas;
- testes automatizados com xUnit;
- Docker e Docker Compose;
- deploy do frontend e backend no Render;
- README técnico com prints e instruções completas.
